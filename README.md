# Ecommerce 
> E-Commerce app UI design in flutter

## :scroll: Description
Ecommerce App Allows you to view list of items and details of a particular item. The app is build  in getX package.


## :bulb: Motivation and Context
<!--- Optionally point readers to interesting parts of your submission. -->
   App has following screens :
	
	1 ) Splash screen 
	2 ) Login 
	3 ) Register 
	4 ) Forgot password 
	5 ) Home 
	6 ) Offers
	7 ) Cart management 
	8 ) Setting 
	9 ) Item detail 
	10 ) Checkout 
	
	
	This is fully used Gex Packagae.
